# Tech Stack

- Bun workspace monorepo.
- Turbo for orchestration.
- Next.js App Router for `apps/web`.
- Hono + tRPC for `apps/api`.
- Tailwind CSS v4 for styling.
- shadcn/ui-standard component patterns for the design system.
- Prisma schema ownership in `packages/db`.
- Paystack for primary NGN donations.
- Lemon Squeezy for global checkout support.
- Shared payment helpers in `packages/payments`.
